# Gestion_etudiants.c
Système de gestion des étudiants développé en langage C.  Cette application console permet d'ajouter, modifier, supprimer et rechercher des étudiants, gérer leurs notes, calculer la moyenne et sauvegarder/charger les données à partir de fichiers.

Fonctionnalités principales :
- Ajouter un étudiant
- Modifier les informations d'un étudiant
- Supprimer un étudiant
- Rechercher un étudiant par ID
- Afficher la liste des étudiants
- Gérer les notes des différentes matières
- Calculer la moyenne d'un étudiant
- Sauvegarder et charger les données avec des fichiers

Ce projet a été réalisé dans le cadre de l'apprentissage du langage C et de la gestion de la mémoire dynamique ainsi que de la manipulation des fichiers.
